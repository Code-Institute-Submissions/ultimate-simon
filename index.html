The Rules To Simon Are Very Straight Forward. The Idea Of The Game Is To Repeat The Sequence Of Lights That Are Shown, By Clicking Them In The Same Sequence When Prompted To Repeat.

You Win The Game By Repeating Back The Complete Sequence, Set By The No Of Rounds Button At The Bottom Left Of The Home Page.

If The Direction Button At The Bottom Centre Of The Page Is Set To "Fwd", You Repeat The Buttons Back In The Same Sequence. To Make The Task Harder, This Button Can Be Set To "Rev", So You Have To Repeat Them Back In Reverse Order.

If You Want More Of A Challenge Still, The Number Of Buttons Can Be Increased From 4 Buttons To 6 Buttons.
